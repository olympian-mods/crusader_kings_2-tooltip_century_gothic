Tooltip: Century Gothic Bold 18


DESCRIPTION
-----------

[h1]Mod Description[/h1]
This mod changes the tooltip font to "Century Gothic Bold 18", the same font that is used in Stellaris.

[h1]Ironman[/h1]
This mod is Ironman compatible.

[h1]Additional Mods[/h1]
In addition to this mod you can subscribe to the following mods:
[list]
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=735130352]Interface: Century Gothic[/url]: change all fonts to "Century Gothic"
[*] [url=http://steamcommunity.com/sharedfiles/filedetails/?id=739149045]Interface: Stellaris Colors[/url]: Stellaris color palette
[/list]
